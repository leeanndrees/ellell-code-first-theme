## What Makes up a Theme
> A WordPress Theme is a collection of files that work together to produce a graphical interface with an underlying unifying design for a weblog. [...] A Theme modifies the way the site is displayed, without modifying the underlying software. — The [WordPress Codex](https://codex.wordpress.org/Using_Themes)
WordPress Theme Files are usually: - PHP - CSS - JavaScript Some themes also include images, fonts, and other assets.
Most WordPress themes have these files: - index.php - screenshot.png - style.css - functions.php _(optional)_ Most themes have many more, too!

PHP Pro-Tips

  • PHP code needs to be enclosed in opening <?php and closing ?> tags
  • Decide on single or double quotation marks and be consistent*

## More Fun with PHP

In index.php, replace this:

						
the_content();

...with this:

						
the_excerpt();
Refresh your homepage. What happened? [More about the_excerpt()](https://developer.wordpress.org/reference/functions/the_excerpt/)
By default, the WordPress excerpt ends with [...] and doesn't link anywhere. Let's change that.

In functions.php, remove the // in front of lines 36-39, so you have the following:

						
// Custom Excerpt
function ellelltheme_excerpt($more) {
  return ' &hellip; <a href="'.get_permalink().'">Continue Reading &rarr;</a>';
}
add_filter('excerpt_more', 'ellelltheme_excerpt');
